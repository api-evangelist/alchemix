# Alchemix Plans

Alchemix does not operate a traditional SaaS pricing model. The protocol is
fully on-chain and permissionless — anyone can interact with the smart
contracts directly without an account, subscription, or API key.

## Protocol Access

| Tier | Description | Cost |
|------|-------------|------|
| Open Protocol | Direct smart contract interaction on Ethereum, Optimism, and Arbitrum | Free (gas fees only) |
| Frontend App | Use the official app at app.alchemix.fi | Free |
| Subgraph Queries | GraphQL queries via The Graph decentralized network | Query fees in GRT |
| DeFiLlama API | Protocol analytics and TVL data | Free (no rate limits for normal use) |
| CoinGecko API | Token price data (ALCX, alUSD, alETH) | Free tier available; Pro plans available at coingecko.com/api |

## Fee Structure (On-Chain)

- **Borrowing:** 0% interest on loans
- **Yield:** Protocol retains a percentage of yield generated (revenue model)
- **Transmuter:** No fee; gradual exchange of alAssets for underlying

## V3 Planned Parameters

- 90% Loan-to-Value (LTV) ratio
- Meta-Yield Tokens (MYT)
- Fixed redemption mechanisms

For governance and protocol fee parameters see: https://forum.alchemix.fi
